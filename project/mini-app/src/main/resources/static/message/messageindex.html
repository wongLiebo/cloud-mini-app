理赔流程

  

索赔材料：

保险金申请人向保险人申请给付保险金时，应提供如下材料：

（1）保险金给付申请书

（2）申请人的有效身份证件

（3）支持索赔的全部账单、证明、信息和证据，包括但不限于医院出具的病历资料、医学诊断书、处方、病理检查、化验检查报告、医疗费用原始单据、费用明细单据等。保险金申请人因特殊原因不能提供上述材料的，应提供其它合法有效的材料。

（4）申请人所能提供的其他与确认保险事故的性质、原因等有关的其他证明和资料。

（5）若保险金申请人委托他人申请的，还应提供授权委托书原件、委托人和受托人的身份证明等相关证明文件。

保险金申请人未能提供有关材料，导致保险人无法核实该申请的真实性的，保险人对无法核实部分不承担给付保险金的责任。

在保险人的理赔审核过程中，保险人有权并在合理的范围内对索赔的被保险人进行医疗检查。此外，保险人应有权在法律允许情况下，要求尸检检验。此类检验费用由保险人承担。在拒赔的情形下，保险人将承担因投保人提供索赔要求所必需的证明、收据、信息和证据而产生的费用。

常见问题

  • 本产品的投保年龄是多少？

    首次投保本产品的客户，年龄限制为：出生满30天-60周岁。本合同期满，投保人可向保险人申请续保本合同，投保人续保本合同须经保险人审核同意，最高可续保至80周岁。

  • 在哪些医院接受治疗可以获得理赔？

    被保险人因意外伤害或者疾病在中华人民共和国境内（不含港、澳、台地区）二级及以上公立医院普通部接受住院、住院前7天后7天门急诊、特殊门诊、门诊手术治疗的，我们报销您需个人支付的、必需且合理的医疗费用。

  • 本产品的等待期是多久？

    被保险人首次投保或非续保时，本合同约定的重大疾病等待期为60天，但甲状腺、扁桃体、疝气、女性生殖系统疾病的检查与治理等待期为90天，其他责任等待期为30天，意外医疗及续保无等待期。

  • 之前购买的其他公司医疗险产品即将到期，今年购买本产品，会有等待期吗？

    被保险人持其他保险公司医疗险保单（指保险责任包含但不限于一般住院医疗费用和恶性肿瘤、重大疾病住院费用且保额100万及以上的），无任何理赔记录，且在投保本产品时，被保险人符合本产品的健康告知，其已经度过的等待期天数可累积到本产品约定的等待期内计算。

  • 等待期的减、免政策如何规定的？

    被保险人持其他保险公司医疗险保单（指保险责任包含但不限于一般住院医疗费用和恶性肿瘤、重大疾病住院费用且保额100万及以上的），无任何理赔记录，且在投保本产品时，被保险人符合本产品的健康告知，其已经度过的等待期天数可累积到本产品约定的等待期内计算。

  • 年免赔额1万元，怎么理解？

    本产品免赔额为每人1万元/年，指在本合同保险期间内，应由被保险人自行承担，本合同不予赔付的部分。社保个人账户支付的金额和通过其他途径(包括被保险人的工作单位、保险人在内的任何商业保险机构)已报销赔付部分可计入免赔额，但社保统筹或公费医疗报销部分不能计入免赔额。若被保险人在等待期后经医院初次确诊罹患本合同约定的重大疾病且在医院接受住院治疗的，对于自确诊重大疾病之日起所发生的符合本合同约定的全部医疗费用，保险人在给付保险金时，不再扣除免赔额。

  • 所有的费用都可以报销吗？

    鼎和致力于提供人人买的起的医疗保险，等待期后被保险人因意外伤害或者疾病在中华人民共和国境内（不含港、澳、台地区）二级及以上公立医院普通部接受住院、住院前7天后7天门急诊、特殊门诊、门诊手术治疗的，我们按照合同约定报销您需个人支付的、必需且合理的医疗费用。

  • 哪些情况可以没有免赔额？

    若被保险人在保险期间内初次确认罹患恶性肿瘤+条款约定的5种重大疾病且接受住院治疗的，对于自确诊之日起剩余保单年度内发生的所有医疗费用，在计算保险金时不再扣除免赔额。

  • 门诊费用可以报销吗?

    门诊手术、住院前7天后7天的门急诊、特殊门诊（恶性肿瘤化学疗法、放射疗法、免疫疗法、内分泌疗法、靶向疗法的治疗费用以及器官移植后的门诊抗排异治疗费用）费用均可报销。保险期间最高报销金额和住院费用合计300万元，如果因恶性肿瘤+条款约定的5种重大疾病接受治疗最高报销金额和住院费用合计600万元。除上述门诊，其他情况在门诊治疗的费用仍需要被保险人自己支付。

  • 投保前已经生的病可以报销么？

    不可以。为了使大家可以用最优的保费获得最高的保障，在发生疾病时真正获得高额医疗费用报销，本险种不接受带病投保的行为。首次投保前已患有的疾病以及症状，均不属于保障范围。
